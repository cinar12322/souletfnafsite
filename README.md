# Soulet — Oyun Tanıtım Sitesi

Soulet oyununun resmi tanıtım ve landing page sitesi.

## Tech Stack
- Next.js 16 (App Router) + TypeScript + TailwindCSS v4
- NextAuth v5 (Google, Discord, Credentials)
- Prisma + Supabase (planlanıyor)
- Vercel deploy (planlanıyor)

## Kurulum
```bash
npm install
npm run dev -- -p 3001
```
